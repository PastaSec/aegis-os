# AEGIS Capability Specification

Every feature inside AEGIS is a Capability.

Capabilities communicate through shared state.

Capabilities never communicate directly with each other.

Current Capabilities

* Dashboard
* Knowledge
* Community
* Notes
* Monitoring
* Hardware
* Network
* Settings

Future

* Weather
* GPS
* LoRa
* APRS
* Mesh
* Cameras
* SDR
* Drone

Capabilities should expose services instead of UI whenever possible.
