# FEMA Preparedness Knowledge Pack

Status: curated

This pack contains field-focused preparedness references curated from FEMA and Ready.gov public guidance.

## Curation Notes

- Documents are summarized and normalized for offline field use.
- Source URLs are preserved in document metadata.
- Markdown documents remain readable without AEGIS.
- Runtime currently supports top-level `docs/*.md` files only.
